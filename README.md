## Spring 게시판 CRUD API

Spring 을 이용하여 제작한 게시판 REST-API 입니다.    
모든 API 에는 상태코드 및 메시지가 반환됩니다.    
테스트의 경우 PostMan 과 테스트 코드를 이용하여 진행하였습니다.    

---
- 작업환경
	- IDE: Intellij
	- D B: MySQL
	- Mapper: JPA
- 주요기능
	- 전체 조회
    - 단일 조회
    - 생성
    - 수정
    - 삭제

어플리케이션을 가동 후 http://localhost:8080/swagger-ui.html 에 접속하여 Swagger api 문서를 확인할 수 있습니다.
